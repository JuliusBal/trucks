- Laravel v10.32.1

### Installation:
1. composer install
2. npm install
3. create DB.
4. cp .env.example .env (enter your DB credentials)
5. php artisan key:generate 
6. php artisan migrate
7. php artisan serve
8. (optional) php artisan db:seed --class="TruckSeeder"
